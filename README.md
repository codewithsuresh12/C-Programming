## History of C Programming

C was developed by **Dennis Ritchie** at **AT&T Bell Laboratories** in **1972**.

**Key Milestones:**
1. **1960**: ALGOL language introduced
2. **1967**: BCPL developed by Martin Richards
3. **1970**: B language created by Ken Thompson
4. **1972**: C language developed by Dennis Ritchie as an improvement over B
5. **1978**: "The C Programming Language" book by Kernighan & Ritchie published - Known as "K&R C"
6. **1989**: ANSI C standard established - C89/C90
7. **1999**: C99 standard released with new features
8. **2011**: C11 standard released

C was originally developed to re-implement the **UNIX operating system**. It is known as the **"Mother of all programming languages"** because many modern languages like C++, Java, and Python are influenced by C.

## Variables in C

A **variable** is a named memory location used to store data that can be modified during program execution.

**Rules for naming variables:**
1. The first character must be a letter or underscore `_`.
2. Digits are allowed but not as the first character.
3. Spaces and special symbols like `@, #, $` are not allowed.
4. Reserved keywords such as `int, float, return` cannot be used.
5. C is case-sensitive: `age` and `Age` are different variables.
